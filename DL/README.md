## Macro Sales Forecasting

**Time series model predicting sales based on macroeconomic variables.**

Development of an application forecasting sales for the next year or estimating sales based on user-defined parameters.

Dataset: https://www.kaggle.com/datasets/yasserh/walmart-dataset


**A Multilayer Perceptron (MLP) regression model was used for predicting continuous values (sales). The model included several hidden layers, the Adam optimizer, and the Mean Squared Error (MSE) loss function.**
